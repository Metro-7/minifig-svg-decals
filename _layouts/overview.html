Overview

The figures directory contains completed designs.

The library contains reusable parts like clothing, torsos and logos.

You can find templates and an inkscape/gimp color palette in resources.

Attributions

  • The minifig template is from minifigcustomizationnetwork.com
  • The microfig template is from Eurobricks.com
  • The palette is based on color tables from Peeron.com and Eurobrick's member Imperial Shadow.
